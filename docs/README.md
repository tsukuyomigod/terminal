# Terminal Backend

In this project, I have created the logic of the terminal for better understanding of low level.

## Tech Stack

**Client:** qt

**Server:** C++

**Libraries** : system api

## Installation

Install from my git repo

```bash
git clone abc
```

## Usage

```bash
cd terminal_gui/build
make
cd ../bin
./terminal-project
```
